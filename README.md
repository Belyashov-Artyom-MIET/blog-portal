# Blog Portal - СМИ портал блогов

Учебный проект по дисциплине "Программная инженерия" (магистратура, РТУ МИРЭА)

## Описание проекта

Веб-портал для публикации и обсуждения статей на заданную тематику с системой аутентификации, управления контентом и комментирования.

### Основной функционал:
- Регистрация и аутентификация пользователей
- Создание, редактирование и публикация статей
- Система комментариев и модерации
- Поиск и фильтрация контента по категориям и тегам
- Личный кабинет автора

## Технологии

- **Project Management:** GitHub Issues + Projects

## Управление проектом

Проект разработан с применением методологии **ScrumBan** (гибрид Scrum и Kanban).

### Артефакты проекта:
- **Epics** - крупные функциональные области
- **Features** - модули системы
- **PBIs** (Product Backlog Items) - пользовательские истории
- **Tasks** - технические задачи
- **Bugs** - ошибки и дефекты

### Спринты:
- **Sprint 1** (MVP Ядра) - базовая функциональность
- **Sprint 2** (Расширенный функционал) - дополнительные возможности
- **Sprint 3** (Релиз) - Релиз версии 1.0.0
